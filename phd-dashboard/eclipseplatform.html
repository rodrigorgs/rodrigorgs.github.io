Bug episodes

We consider that, every time a bug is reopened, a new bug episode begins. Therefore, most bugs have only one episode, a bug, bugs that are reopened once have two episodes, and so on. We can also classify episodes in reopening or non-reopening. The first episode of a bug is also non-reopening. If a bug has more than one episode, all episodes except the first are reopening.

So, first, let's consider only resolved episodes, comparing those that are reopenings with those that aren't. The next table shows summaries for the number of days needed to resolve the bug.

mean median sum
regular 77.2 7.1 2523064.5
REOPENED 64.0 2.5 235144.2
=> ratio (%) 82.80% 35.69% 9.32%
